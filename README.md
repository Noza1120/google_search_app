# 概要

キーワードを入力し、Google検索結果を一覧表示させるアプリ

公開URL: https://google-search-app-v1.herokuapp.com/

## 機能要件

* ユーザが文字列入力フォームにキーワードを入力し、入力したキーワードでのGoogle検索結果を一覧表示すること

## 非機能要件

* Google 検索結果の取得には、「Custom Search JSON API」を使用すること

## 使用技術

Ruby on Rails
